Godzi Examples

Load Map - Demonstrates the basics of loading a map
Sky - Demonstrates how to show the sky and set the date/time
Layers - Demonstrates the basics of adding and removing image and elevation layers
Annotations - Demonstrates the basics of annotations
KML - Demonstrates adding a KML file to the map
Models - Demonstrates the basics of loading and manipulating models
Events - Demonstrates subscribing to and responding to an event
GeoRSS - Demonstrates how to load a GeoRSS feed and provide custom rendering
GPX - Demonstrates how to load a GPX file as well as how to place text labels
